## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/apexcut.git
```

2. Navigate to the project directory:
```bash
cd apexcut
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

5. Open your browser and visit:
```
http://localhost:5173
```
